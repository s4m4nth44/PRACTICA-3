# Práctica 3 - Display de 7 Segmentos con PIC16F887

## Objetivo
El manejo de un display de 7 segmentos mediante el microcontrolador PIC16F887 para desplegar secuencias numéricas.

## Software utilizado
- MPLAB X IDE
- Compilador XC8
- Proteus 8

### Clase: Contador de 0 a 9
Implementación de un contador decimal de 0 a 9 utilizando un display de 7 segmentos controlado por el PIC16F887.

### Actividad 1: Contador hexadecimal de 0 a F
Implementación de un contador hexadecimal de 0 a F utilizando un display de 7 segmentos controlado por el PIC16F887.
